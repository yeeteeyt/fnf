Friday Night Funkin': Foned In

Friday Night Funkin': Foned In

mobile browser and touch screen support because beep bo bop

(April 21st 2021 - Not adding Week 7 yet, read here.)

On smaller devices, I'd recommend using downscroll as to not block the notes with your fingers. Old Input has the anti-mashing system built in, while New Input lets you mash away. Your device should have 2+ (ideally 4+) GB of RAM so that levels won't crash on restart.

Controls (In Menus):

  • Navigate Up = Swipe Down
  • Navigate Down = Swipe Up
  • Change Difficulty = Swipe Right
  • Go Back = Swipe Left
  • Enter/Select = Tap

Controls (In Game):

  • Touch the arrow keys on your side as notes reach the judgement line
    • You can also touch anywhere in the same column if it's easier
  • Tap the top-left corner to pause the game

Supported Browsers (Latest Version):

  • Safari
  • Chrome

Turn off auto-rotate and rotate your device to play in landscape. 

Known Issues:

  • Restarting songs might reload your page
  • Possible stuttering and hitches
    • Potential fix is to restart that level
  • Sprites might not work on really old devices
  • May crash for no reason at any given time
  • Accuracy bar still not accurate (xd)
  • Other funky glitches with Kade Engine

Credits:

Special thanks to Kade Dev for creating the modified Friday Night Funkin' engine with revamped input. Please support the official devs as well by dropping a few bucks if you enjoyed the game!

The source code is available on my Github, but fair warning your eyes may bleed after reading it. Foned In was built using Kade Engine 1.3.1 with some modifications made.
